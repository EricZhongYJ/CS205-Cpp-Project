## 2. 代码运行

对于文件数量为1的项目，可以使用命令行来编译并运行，如(Project1中)：

```bash
g++ source.cpp -o mul & mul 1 2
```

对于文件数量超过1的文件可以使用`cmake`来编译，如(Project2中)：

```bash
cmake . & make
./Project2.out
```
